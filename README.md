# MangaNest 📚

**MangaNest** is a lightweight, high-performance Android application designed for reading Manga and Manhwa. Built natively in Java, it focuses on a clean user experience, fast image loading, and a seamless reading flow.

## 🚀 Features

  * **Sleek UI/UX:** A modern interface designed for mobile-first reading.
  * **Custom Splash Screen:** A polished entry point to the app experience.
  * **Optimized Performance:** Efficient handling of high-resolution manga pages.
  * **Native Android Integration:** Built using Java and Android Studio for maximum stability.


## 🛠️ Tech Stack

  * **Language:** Java
  * **IDE:** Android Studio
  * **Version Control:** Git / GitHub
  * **Build System:** Gradle

## 📥 Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/YOUR_USERNAME/MangaNest.git
    ```
2.  **Open in Android Studio:**
    Select "Open an existing project" and navigate to the cloned folder.
3.  **Build and Run:**
    Click the **Run** button or press `Shift + F10` to deploy to your emulator or physical device.
